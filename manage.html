upload

You can import your contacts into Dash using the form below. The import file must be in comma separted value (CSV) format which you can easily generate from an Excel, Numbers, OpenOffice, LibreOffice, or Google Spreadsheets file using their built-in export functionality.

Please see the import format example file for a reference on how your import file should be formatted.

  • Instructions

  • 1. Download the sample form here (hyperlink to .csv file)
  • 2. The only required sections are [your user name here], [your password here], and Student First Name. If all you have is a roster of students with their full names, paste that into the Student First Name column.
  • 3. Or, fill out more! As much or as little as you like.
  • 4. Please limit phone types for Emergency Contacts to work, home, and cell or this upload won’t work.
  • 5. Save or export as a .csv
  • 6. Attach it through Choose File and hit Import!

how to...

creating your account and adding students

Create your account when you launch Dash. You can make as many accounts as you like from your existing email addresses - Make sure to hang on to your password. After that, use our upload system to add students, or manually add them through the Roster.

adding guardians

In each student, hit the plus sign to add a Guardian (you need good aim for this!). Use the “Relationship” field to add the relation and any other short notes you might find helpful. For example, “Mom, after 5” can help remind you the best time to contact a parent. Later, use the edit button on the student’s Contact Info tab to edit or add contacts.

tagging students

In your Roster, tap a frown or smile to add a student to your Playlist. Swipe the student if you want to add a subject or reminder for why you tagged them. Siri can help too!

playlist and calling parents

The Playlist will automatically pull the best phone number for you to call. If it is your first time calling, it will cycle your numbers to help you find the correct one. If you want to reach out to a parent you did not add to the playlist, you can call, text, or email from an individual student’s Contact Info tab.
